# homebrew-noai-watermark

Homebrew tap for [noai-watermark](https://github.com/mertizci/noai-watermark).

## Install

```bash
brew install mertizci/noai-watermark/noai-watermark
```

## Update

```bash
brew upgrade noai-watermark
```

Formula is automatically updated on each [noai-watermark release](https://github.com/mertizci/noai-watermark/releases).
